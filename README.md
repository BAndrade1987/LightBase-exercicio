# LightBase-exercicio

## Descrição
Projeto desenvolvido para processo seletivo LightBase.  
Trata-se de uma **API de clientes** construída com **FastAPI** e **PostgreSQL**, empacotada com **Docker** e orquestrada via **Docker Compose**.  
Permite criar, consultar, atualizar e deletar clientes, além de buscar clientes pelo final da placa do carro.

## Pré-requisitos
- Docker: [https://www.docker.com/get-started](https://www.docker.com/get-started)  
- Docker Compose: [https://docs.docker.com/compose/install/](https://docs.docker.com/compose/install/)  
- Navegador ou ferramenta para testar APIs (ex.: Postman, Insomnia, curl)
